# Student Performance Analysis System

## Description
This project analyzes student academic performance using Python.
It helps in understanding subject-wise performance, pass/fail ratio,
and overall trends using real-world student data.

## Technologies Used
- Python 3.11
- Pandas
- Matplotlib
- CSV Dataset

## Features
- Load and process student performance data
- Analyze marks and results
- Generate visual graphs
- Real-world dataset analysis

## How to Run the Project
1. Install Python 3.x
2. Install required libraries:
   pip install -r requirements.txt
3. Run the analysis file:
   python src/analysis.py

## Output
- Graphical representation of student performance
- Insights from the dataset
